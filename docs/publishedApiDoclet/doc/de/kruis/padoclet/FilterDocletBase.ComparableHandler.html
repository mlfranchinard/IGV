    public static class FilterDocletBase.ComparableHandler
extends FilterDocletBase.HandlerBase
    This class is the base of all the HalfDynamicProxy classes for the javadoc *Doc interfaces. All com.sun.javadoc.*Doc interfaces extend the Comparable interface. Additionally instances of these interfaces must be comparable by reference. Therefore this class contains a matching compareTo(Object) implementation.
