This packages contains javadoc examples only.

The main purpose of this package is to provide you with some examples for the usage of the PublishedApiDoclet tags. Please be aware, that these examples currently do not fully cover the posibilities of PublishedApiDoclet. Extentions are highly welcome.

Note: Every @pad...-tag within this package and its subpackages carries the sample label. Therefore I can use the --padDefaultIsExclude option, to exclude the sample from the javadoc documentation of the real source of PublishedApiDoclet. To format the sample documentation, I use --padFilterDefault sample.
